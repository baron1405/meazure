enum MeaSection

The data display is divided into two major sections: Region and Screen.

The Region section displays the measurement data from the radio measurement tools (e.g. Point tool, Line Tool). The Screen section displays data about the current monitor (e.g. resolution, width, height). These identifiers are used to identify a data display section for enabling and disabling purposes.
