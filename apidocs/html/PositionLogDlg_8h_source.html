28/// A modeless dialog for recording new positions and managing positions
29/// that have already been recorded. Besides recording positions, the
30/// dialog allows individual positions to be deleted, replaced, and
31/// annotated.
32///
33/// This class is an observer of the position log manager. This allows
34/// the dialog to be called when positions are recorded, deleted,
35/// loaded or saved.
36///
