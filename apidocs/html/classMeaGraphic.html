Detailed Description

Base class for all graphic elements.

Classes derived from this base class are used by the measurement tools to perform their function.

bool MeaGraphic::Create ( LPCTSTR  classname,
const SIZE &  size,
const CWnd *  parent,
UINT  id = 0xFFFF 
)

Creates the window for the graphic.

This method must be called before the graphic can be displayed and before any operation that attempts to manipulate the graphic's window.

Parameters
classname[in] Class name for the window. Typically, a derived class obtains a window class name from a call to the AfxRegisterWndClass function.
size[in] Initial size for the window.
parent[in] Parent window or NULL if the graphic does not have an immediate parent. If a parent window is specified, the graphic is created as a child window (WS_CHILD). Otherwise the graphic is created as a popup window (WS_POPUP) and is assigned the main window as a parent.
idUnique ID for this graphic element. Typically, the ID is used to identify the graphic element in a callback.
Returns
true if the window is created successfully.

References m_id, and m_parent.
