Detailed Description

Point measurement tool.

This tool provides a crosshair that can be placed arbitrarily on the screen to measure a location.

◆ Strobe()

void MeaPointTool::Strobe ( )
virtual

Visually strobes the tool.

The tool crosshair is cycled between a highlight color and its normal color once. In addition, the data window attached to the crosshair is cycled between hidden and visible. Strobing the tool is done to visually confirm position recording.
