Detailed Description

The view class for the application.

There is only a single view in this application and it is owned directly by the application frame window.

◆ LoadStartupFile()

void AppView::LoadStartupFile ( const CString &  cmdLineFile) const

This method handles the processing of any files specified on the command-line or specified in preferences.

The following processing is performed:

  1. If a profile file is specified on the command-line, that file is loaded.
  2. If no file is specified on the command-line, or the file is not a profile, the preferences are checked to see if a startup profile file has been specified. If it has, that file is loaded.
  3. If a position log file is specified on the command-line, that file is loaded.
Parameters
cmdLineFile[in] File pathname specified on the command-line.
