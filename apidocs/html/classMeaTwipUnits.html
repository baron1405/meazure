Detailed Description

Provides linear measurements in twips.

Member Function Documentation

◆ FromPixels()

FSIZE MeaTwipUnits::FromPixels ( const FSIZE &  res) const
protectedvirtual

Returns the X and Y factors to convert from pixels to twips.

In other words, multiplying the values returned from this method by pixels results in a value expressed in twips.

Parameters
res[in] Screen resolution, in pixels/inch.
Returns
X and Y conversion factors, in twips/pixels.
