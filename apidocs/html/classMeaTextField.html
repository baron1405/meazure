Creates the window for the field.

To make the field visible, add the WS_VISIBLE style or call ShowWindow.

Parameters
style[in] Field styles as defined for CEdit.
topLeft[in] Position of the top, left corner for the field, in pixels.
numChars[in] Number of visible characters. For proportional fonts the number of characters actually visible may vary from the specified value.
parentWnd[in] Parent window for the field.
id[in] ID for the field.
Returns
true if the field was created successfully.
