Detailed Description

Provides SAX style XML parsing.

This class is a wrapper around the expat parser and adds XML validation. A class that wants XML parsing services mix-in inherits from the MeaXMLParserHandler class, overrides the methods of that class for the events of interest, creates an instance of this class and points it at the XML file to parse. The parser calls back using the methods of the MeaXMLParserHandler class to indicate various XML parsing events. This is classic SAX parsing behavior. The class can also build a primitive DOM.
