Detailed Description

Mix-in class for XML parser callback methods.

A class that wants XML parsing services mix-in inherits from this class, overrides the methods of this class for the events of interest, creates an instance of the MeaXMLParser class and points it at the XML file to parse. The parser calls back using the methods of this class to indicate various XML parsing events. This is classic SAX parsing behavior. The class contains noop implementations for the method.
