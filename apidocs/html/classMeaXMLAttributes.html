Detailed Description

The class contains the attributes associated with an XML start element.

In addition to iterating through the attributes, the class provides searching and other attribute manipulation capabilities.

Constructs an instance of the class based on the specified attributes provided by the expat parser.

Parameters
atts[in] Array of attribute name/value pairs from expat. In the array the name is followed by the value which is then followed by another name and so on until a NULL is encountered.
numSpecified[in] Number of attributes in the array that have been explicitly specified versus set by default from the DTD. The atts array is organized such that the explicitly specified attributes come first.
