36/// The data display is divided into two major sections: Region and Screen.
37/// The Region section displays the measurement data from the radio measurement
38/// tools (e.g. Point tool, Line Tool). The Screen section displays data about
39/// the current monitor (e.g. resolution, width, height). These identifiers are
40/// used to identify a data display section for enabling and disabling purposes.
41enum MeaSection {
42 MeaRegionSection, ///< Measurement tool data display section.
43 MeaScreenSection ///< Screen data display section.
44};

322 /// possibly a spin control, and usually a units label. This class packages
