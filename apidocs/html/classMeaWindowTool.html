Detailed Description

Window measurement tool.

This tool tracks the mouse pointer and attempts to identify the deepest window in the window hierarchy under the pointer. the dimensions of that window are displayed. The tool consists of a rectangle that is used to highlight the window selected for measurement. In addition, a data window is attached to the rectangle to provide a measurement information popup.
