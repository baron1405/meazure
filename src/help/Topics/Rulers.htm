To assist in measuring items on the screen, a set of horizontal and vertical rulers can be displayed. By default, the rulers will be displayed on the top and left edges of each display screen. The position of the rulers can be changed by dragging them across the screen using the mouse. To aid in measurement, the rulers can be made partially transparent using the ruler preferences.
Rulers can be used in conjunction with any other tool including the screen grid. The rulers provide indicators that show the position of the current measurement tool. For example, when the Cursor tool is selected, the ruler indicator will show the current position of the mouse cursor.
To select the rulers choose the Rulers item on the Tools menu or click on the rulers icon on the toolbar.
