The layout of the Meazure program window can be customized to display only the information you require. Each section of the program window can be individually hidden to reduce the visual footprint of the program.
The following items can be selected from the View menu and toggle the display of the respective sections. Meazure remembers which sections have been hidden so that they remain hidden in subsequent runs of the program.
Toolbar - toggles the display of program toolbar.
Tool Info - toggles the display of the tool information section.
Screen Info - toggles the display of the screen information. section.
Magnifier - toggles the display of the magnifier window.
Status Bar - toggles the display of the status bar.
Collapse/Expand - Hides all sections to provide a minimal visual footprint. If hidden, restores the original section layout.
Note that the menu bar cannot be hidden.
