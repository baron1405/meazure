There are four methods for making fine adjustments to the position of a crosshair. The first method is to use the up/down buttons adjacent to the position text fields. These change the associated position by a one pixel per click. The second method is to press Ctrl-1, Ctrl-2 or Ctrl-3 and an arrow key. The number represents the position to move. The third method is to enter a position value directly into the position text fields. The fourth method is to use the magnifier at a high magnification while dragging the crosshair to the desired location.
