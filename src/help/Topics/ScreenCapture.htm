The Rectangle tool, Circle tool, and the Window tool provide the ability to capture the contents of the screen. Both tools copy the image they capture to the clipboard. Once copied to the clipboard, the image can be pasted into any application that accepts image data such as the Windows Paint tool or Adobe Photoshop®.
The Rectangle tool allows you to capture arbitrary portions of the screen to the clipboard. Note that the area captured is equal to the full width and height of the rectangle. Therefore, position the rectangle within the edge of the area to be captured.
The Circle tool allows you to capture arbitrary portions of the screen to the clipboard. Note that the area captured is equal to the width and height of the square bounding the circle.
The Window Tool allows you to capture the contents of top level program windows and their child windows.
To perform a capture, select either the Rectangle or Window tool then select the Copy Region item on the Edit menu or press Ctrl+R.
