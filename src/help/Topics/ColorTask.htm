The bottom portion of the magnifier contains a color readout. This readout consists of a color swatch and color value. The color represents the pixel under the cursor or at the center of the crosshair depending on the current measurement tool. The color value can be copied to the clipboard using the button with the clipboard icon immediately to the right of the color swatch.
The color of a pixel can be matched to one of the web extended or web basic colors defined by the W3C. See the description of the color matching formats for details.
